## 로컬 개발 환경 세팅

### npm 종속성 라이브러리 설치
```
npm install
```

### 실행
소스코드 컴파일 및 코드 수정 시 빠른 리로드가 가능한 개발 버전
```
npm run serve
```

### run 버튼 설정 (CLI 명령어 없이 실행)

1. Edit Configuration
2. npm
    - package.json: `~\dealtype-worldcup-vue\package.json`
    - Command: `run`
    - Scripts: `serve`
    - Node interpreter: `node.exe 20.15.0`
    - Package manager: `npm.cmd 10.8.3`
3. run
